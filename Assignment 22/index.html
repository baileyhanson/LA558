turf.nearest - Takes a reference point and a FeatureCollection of Features with Point geometries and returns the point from the FeatureCollection closest to the reference. This calculation is geodesic.

turf.distance - Calculates the distance between two points in degrees, radians, miles, or kilometers. This uses the Haversine formula to account for global curvature.

turf.buffer - Calculates a buffer for input features for a given radius. Units supported are miles, kilometers, and degrees.

turf.lineSlice - Takes a line, a start Point, and a stop point and returns a subsection of the line in-between those points. The start and stop points don't need to fall exactly on the line. (could be useful for slicing sidewalk line segments when accessing condition or availability)

turf.random/turf.sample - Takes a FeatureCollection and returns a FeatureCollection with given number of features at random.

turf.squareGrid - Creates a square grid from a bounding box, Feature or FeatureCollection. (could be useful for walkscore)
